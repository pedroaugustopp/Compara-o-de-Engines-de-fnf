# Compara-o-de-Engines-de-fnf
O Comparação de Engines - FNF é um site front-end interativo (HTML5/CSS3) que analisa as principais engines do jogo (Kade, Psych e V-Slice). Ele detalha inovações técnicas, história e gameplay, servindo como guia prático para modders, com links diretos para os repositórios oficiais no GitHub.
