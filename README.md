# Compara-o-de-Engines-de-fnf
Nome do Projeto: Comparação de Engines - Friday Night Funkin'
Descrição Geral:
Um site front-end interativo e responsivo desenvolvido para apresentar, analisar e comparar as principais engines (motores de jogo) utilizadas pela comunidade de Friday Night Funkin' (FNF). O projeto detalha a história, as inovações técnicas e o estilo de gameplay das três bases mais influentes do jogo: Kade Engine, Psych Engine e V-Slice Engine.

Objetivo:
Informar jogadores e desenvolvedores de mods (modders) sobre as diferenças estruturais, de otimização e de sintaxe de cada versão. O site serve como um guia rápido para ajudar criadores a escolherem a melhor ferramenta para seus próprios projetos, fornecendo também acesso direto ao código-fonte oficial de cada uma.

Tecnologias Utilizadas:

HTML5: Estruturação semântica do conteúdo, organização em blocos lógicos e integração de links externos.

CSS3: Estilização visual moderna. Utiliza fundos com gradientes fixos, efeitos de transição tridimensionais (hover), manipulação de proporção de imagens (object-fit: contain) e tipografia personalizada importada do Google Fonts (Montserrat e Poppins).

Principais Funcionalidades / Destaques:

Design Temático: Identidade visual inspirada na estética vibrante e urbana do jogo original, com cores fortes, fontes robustas e efeitos de sombra estilo "neon".

Cards Informativos: Conteúdo organizado em cartões com profundidade e interatividade, garantindo uma leitura escaneável e agradável.

Imagens Otimizadas: Sistema de contêineres que garante que as capturas de tela das engines sejam exibidas por inteiro, sem cortes ou distorções.

Acesso ao Código-Fonte: Títulos integrados com links que redirecionam o usuário diretamente para os repositórios oficiais do GitHub de cada desenvolvedor.
